# 1. Update packages and install git
```sudo apt update```

```sudo apt install -y git```

# 2. Clone the installer repo
```git clone https://github.com/WaterIsOkIGuess/PiHole-Installer-Script.git```

# 3. Go into the repo
```cd PiHole-Installer-Script```

# 4. Make sure the installer is executable
```chmod +x pihole-install.sh```

# 5. Run the installer (this will do the Pi-hole setup)
```sudo ./pihole-install.sh```
